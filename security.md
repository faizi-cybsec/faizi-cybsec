# Security Policy

## Supported Versions

This project is currently maintained under the latest stable release.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅         |

## Reporting a Vulnerability

The security of this project is taken seriously. If you discover a security vulnerability or security-related issue, please report it responsibly by opening a private issue or contacting the project maintainer.

Please include the following information when reporting a vulnerability:

* Description of the vulnerability
* Steps to reproduce the issue
* Potential impact
* Suggested remediation (if available)

Thank you for helping improve the security and reliability of this project.
